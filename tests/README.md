# tests

Unit tests for the GatherBrain source code. Mirror the subdirectory structure of `src/`.

## Subdirectories

- **`query/`** — Tests for the query engine and `parseQueryString`.
- **`schema/`** — Tests for built-in schemas, the vault schema loader, and the schema registry.
- **`store/`** — Tests for entity store loading/querying, file-path helpers, and the entity file parser.
